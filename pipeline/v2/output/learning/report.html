How It Works

Cell produces outputColor, shape, or behavior cell generates its artifact blindly
Validator checksBinary gate: valid hex? Renders to pixels? Parses as JS? Pass or fail.
Survived → storedPassing outputs are saved to the memory bank. Failures vanish without trace.
Next run: few-shotFuture cells receive 1-2 random past survivors as "previously accepted" examples
No reason givenThe cell never learns WHY an output was accepted. Only THAT it was. This preserves blindness.
